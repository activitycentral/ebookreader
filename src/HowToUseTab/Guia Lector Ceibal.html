Guía de uso


Este documento detalla las funcionalidades del Lector Ceibal y el correcto uso del mismo.

El Lector Ceibal tiene como fin mejorar la experiencia de lectura de los usuarios, haciéndola la experiencia más interactiva y fácil de usar.


El lector cuenta con dos modos de uso:


  • Modo normal: El usuario utiliza el teclado y el touchpad para navegar y utilizar las funcionalidades del lector.


  • Modo libro: El usuario utiliza las teclas de juego para la lectura del libro. Este modo permite ocultar el teclado y utilizar el gamepad si la laptop es una XO. Si es una Magallanes, puede utilizar las teclas indicadas de su teclado (ver más abajo).




Menú principal


Al ingresar, el usuario encontrará la siguiente barra con herramientas que lo llevarán a diferentes funcionalidades del Lector Ceibal.






Mis Libros


En esta interfaz el usuario podrá ver y administrar su biblioteca personal, donde están los libros que descargó de la Biblioteca Ceibal o agregó manualmente.



Bibliotecas


Esta funcionalidad ofrece al usuario varias opciones para agregar libros a su biblioteca personal. Puede seleccionar un libro o más de uno e importarlos con un solo click.



Leer


Esta opción del menú abre el lector de libros, que permite al usuario leer en los formatos .pdf y .epub.



Ayuda


Esta opción abre esta guía de ayuda para el uso del Lector Ceibal.



Parar


Cierra el Lector Ceibal.




Menú Leer


La función Leer ofrece la siguiente barra de herramientas:





Marcar


Este botón permite agregar una marca en el texto o quitarla si ya hay una. Al agregar una marca aparece en la esquina superior derecha del texto el siguiente ícono:

Estas marcas quedan guardadas cuando se cierra el libro y permiten continuar la lectura en el punto en que se la dejó.

Las teclas de juego permiten navegar entre las marcas ya agregadas.



Tamaño del Texto



El formato .ePub permite que el usuario aumente o reduzca el tamaño del texto mediante estos botones.



Acercarse y Alejarse



El formato .PDF permite que el usuario acerque o aleje la imagen haciendo click en estos botones.



Buscar en el Texto


Esta función permite encontrar en el libro las palabras que se ingresen en el campo de texto. Los botones “Buscar Anterior” y “Buscar Siguiente” permiten navegar por las distintas veces que aparece la palabra en el texto.



Pantalla Completa / Regresar



Este botón lleva la ventana de texto al tamaño de toda la pantalla, ocultando las barras de menú. El botón para regresar a la vista con barras de menú aparece en la esquina superior derecha de la pantalla cuando se mueve el cursor.



Página Anterior / Página Siguiente


Para avanzar y retroceder en el libro se pueden usar las flechas del teclado o la rueda de cursor en las XO.



Teclas de Juego


Estos botones de la XO permiten acceder a funciones del Lector Ceibal.

Cuando el usuario oprime alguna de las Teclas de Juego, aparece en pantalla el siguiente gráfico para informarle cuál fue la opción elegida:


Marca Anterior Lleva a la primera marca que encuentra retrocediendo en el texto.

Marca Siguiente Lleva a la primera marca que encuentra avanzando en el texto.

Agregar/Quitar Marca Agrega una marca en el lugar en que se encuentra el cursor, o la elimina si ya existe una.

Opciones Abre el Menú de Opciones.




Opciones


Este menú ofrece herramientas dentro del Modo Libro, sin necesidad de utilizar el teclado.



Aumentar o disminuir el Tamaño del Texto (sólamente para el formato ePub).


Acercarse o Alejarse de la imagen del libro (sólamente para el formato PDF).


Navegar por Mis Libros:

La tecla de juego abre el libro seleccionado.


La tecla de juego cierra el Menú de Opciones sin cambiar de libro.
